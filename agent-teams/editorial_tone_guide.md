# マチノワ 編集トーンガイド（単店舗特集 / 街ガイド共通）

**策定: 2026-05-15 / 基準記事: feature-qualia-meinohama (Kitchen&Bar Qualia)**

ユーザー承認済みのトーン。今後の特集記事は全てこのトーン基準に沿って執筆する。

---

## 基本方針

ライター本人が、目の前で読者に語りかけているような筆致で書く。
編集部の主観・体感・推奨を、過剰に丁寧すぎない言葉で混ぜ込む。

- ベースは「ですます調」
- 要所で「〜だろう」「〜が最適解」「〜の一択」「〜と思う」などの**断言・体言止め**を混ぜる
- 読者を直接「あなた」呼ばわりはしない（一段引いた編集者目線）
- 「〜してみてください」「〜してください」は使うが、文末で多用しすぎない
- 「〜なんですよね」「〜あるある」など、口語表現も適度に
- 編集部の主観を出すフレーズ:
  - 「編集部の結論です」
  - 「正解だ」
  - 「最適解」
  - 「あるある」
  - 「これって、意外と〜」
  - 「〜じゃないかなと思う」
  - 「ぜひ届けたい一軒」

## 避けるトーン

- 過剰に丁寧な「ですます」尽くし（→ 読み物として単調になる）
- 「〜です。〜です。〜です。」が3文以上続く
- 「〜のです」連打（説明調が重くなる）
- 客観的な紀行文・ガイドブック調（→ 編集部の体温が消える）
- 「だ・である」だけで通すのも NG（記事として硬すぎる）

## 記事タイトルの型

**目的を冒頭に置き、店名と一言キャッチを併記する。**

例:
- 姪浜デートのディナーには**Kitchen&Bar Qualia**。創作イタリアンと本格カクテルで過ごす、大人のための一軒をご紹介
- {地域}{利用シーン}には**{店名}**。{特徴}で過ごす、{ターゲット}のための{何か}をご紹介

避ける表現:
- 「魅力5選」「厳選5」など量で煽る古い言い回し
- 「絶対おすすめ」「日本一」「最強」などの根拠なき断定

## 副題（subtitle）の型

**料理・店舗特徴。{地域}で「{価値の言い切り}」を{動詞}{文末}**

例:
- 創作イタリアンと本格カクテル。福岡市西区・姪の浜で「食事もお酒も1軒で」贅沢を叶えてくれる夜のお店

## 利用シーン別の書き分け（重要）

**※ デートは Qualia の場合に限る。今後の記事は、その店舗の業種・利用シーン・地域に応じて軸を変える。**

| 業種 / シーン | 主軸 | キーワード例 |
|---|---|---|
| 高級ディナー・記念日系 | 大人のデート、特別な日、贅沢 | 大人、記念日、誕生日、特別感、上質 |
| 老舗・名物 | 歴史、地域文化、職人技 | 創業○年、土地の味、伝統、職人 |
| カフェ・喫茶店 | 一人時間、本、午後、読書、静けさ | 一人時間、午後、静か、本、ゆっくり |
| 居酒屋・酒場 | 友達、はしご酒、地元 | 仕事終わり、仲間、ガヤガヤ、地元の名店 |
| ファミリー・大箱 | 家族、子連れ、休日 | 家族、子連れ、週末、ガヤガヤ |
| 観光地・名所 | 旅、初訪問、外せない | 旅、観光、初訪問、外せない一軒 |
| カウンター系（鮨・割烹） | 大人、おまかせ、贅沢 | カウンター、おまかせ、大人、相棒との一献 |
| バー・カクテル | 二軒目、夜、静か | 二軒目、夜更け、静かな夜、深夜 |
| 朝食・モーニング | 朝、休日、爽やか | 朝、休日の朝、目覚め |

地域特性も組み合わせる:
- 観光地（鎌倉、京都、那智勝浦等）: 旅・小旅行・週末 軸
- 都心ターミナル（天神、新宿、梅田等）: 仕事帰り・週末ナイト 軸
- ベッドタウン・住宅街（姪浜、自由が丘等）: 大人の落ち着き 軸
- 海沿い・温泉地: 旅情、休暇 軸

## ranking 部の書き方（POINT 表記の場合）

### desc 短文（〜30字目安）
体言止めで切れ味よく。「〜の一択」「〜が最適解」「〜にこそ届けたい」「〜の一軒」など。

### desc 長文（200〜300字）
1. 周辺の事情・あるある（地元あるある、街の特徴、業種のあるある）から入る
2. その店ならではの解決・特長
3. 編集部の主観1〜2行（「〜なんじゃないかなと感じる」「〜だろう」）
4. 読者へのワンアクション（「覚えておいて損のないポイント」「ぜひ試してほしい」）

---

## quote（編集部の声）
1〜2文。**「〜編集部としてはいちばんおすすめしたい」「〜そこを推したい」**で締める断言フレーズ。

## closing（編集後記）
- 「黄金動線」「編集部の結論」「届けたい一軒」など、編集部の言葉として締める
- 必ず**注意点（失敗しやすい点）**で締める。読者の実用性を担保
- 営業時間・料金は時期で変動するため「訪問前に公式情報をチェックしておくと安心」と入れる

---

## 共通の禁止フレーズ

- 「日本一」「最強」（根拠なし）
- 「絶対」「必ず」（断定が強すぎる）
- 「あなた」（読者を直接呼ばない）
- 「みなさま」「お客様」（記事ではなく店内アナウンス調になる）
- 「ぜひぜひ」「とてもとても」（重複強調はくどい）

## 共通の推奨フレーズ

- 「〜の一択」「〜が最適解」「〜が正解だ」
- 「〜だろう」「〜と思う」「〜と感じる」（編集部の主観）
- 「〜あるある」「地元の人なら知っている〜」（地域目線）
- 「ここぞの夜に」「特別感は十分」（大人系で使える）
- 「届けたい一軒」「覚えておいて損のない」（締めのフレーズ）
